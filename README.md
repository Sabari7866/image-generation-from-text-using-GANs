# image-generation-from-text-using-GANs
# 🧠 Image Generation from Text using Deep Learning

## 📌 Project Overview

This project demonstrates **text-to-image generation** using a pretrained deep learning model.
The system takes a **text description (prompt)** as input and generates a **realistic image** based on the given input.

Instead of training a model from scratch, a **pretrained generative model** is used to produce high-quality outputs efficiently.

---

## 🎯 Objective

* To understand the concept of **text-to-image generation**
* To generate images from natural language descriptions
* To explore modern **generative AI techniques**

---

## ⚙️ Technologies Used

* Python 🐍
* PyTorch
* Diffusers Library
* Transformers
* Matplotlib

---

## 🧩 Model Used

* **Stable Diffusion (runwayml/stable-diffusion-v1-5)**
  A powerful pretrained generative model capable of producing realistic images from text prompts.

---

## 🚀 Features

* Accepts dynamic **user input (text prompt)**
* Generates **high-quality images**
* Supports **GPU acceleration**
* Saves generated output as an image file

---

## 🛠️ Installation

Run the following command to install required libraries:

```bash
pip install diffusers transformers accelerate torch
```

---

## ▶️ Usage

Run the Python script and enter a text prompt when prompted:

```bash
python main.py
```

Example input:

```
Enter your text description: a red sports car on a highway
```

---

## 🖼️ Sample Output

The model generates realistic images corresponding to the given text input.

---

## 📊 Result

The system successfully generates high-quality images from textual descriptions using a pretrained deep learning model. The results demonstrate the effectiveness of generative models in converting text into visual representations.
